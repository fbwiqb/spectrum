🎯 프로그램 소개
원소 방출 스펙트럼 퀴즈는 각 원소의 고유한 스펙트럼 패턴을 학습하고, 혼합된 스펙트럼에서 원소를 식별하는 능력을 키우기 위한 교육용 프로그램입니다.
📚 기본 사용법
1단계: 원소 스펙트럼 관찰
• 화면 왼쪽에 7개 원소(수소, 헬륨, 네온, 나트륨, 수은, 리튬, 크립톤)의 스펙트럼이 표시됩니다.
• 각 원소는 고유한 파장의 빛을 방출하여 독특한 선 스펙트럼을 형성합니다.
• 스펙트럼 아래의 숫자는 파장(nm)을 나타냅니다.

2단계: 퀴즈 문제 생성
• 오른쪽 상단의 버튼을 클릭하여 퀴즈를 시작하세요.
• 방출 스펙트럼: 검은 배경에 색깔 선이 표시됩니다.
• 흡수 스펙트럼: 연속 스펙트럼에 검은 선이 표시됩니다.
• 숫자(2~5개)는 혼합된 원소의 개수를 의미합니다.

3단계: 원소 선택
• 혼합 스펙트럼을 보고 어떤 원소들이 섞여 있는지 추론하세요.
• 아래 원소 버튼을 클릭하여 답을 선택합니다.
• 선택된 버튼은 파란색으로 변합니다.
• 다시 클릭하면 선택이 취소됩니다.

4단계: 정답 확인
• "정답 확인" 버튼을 눌러 결과를 확인하세요.
• 모두 맞으면 🎉 완전 정답 메시지가 표시됩니다.
• 일부만 맞으면 맞춘 개수와 원소가 표시됩니다.

5단계: 정답 보기
• "정답 표시" 버튼을 누르면 왼쪽 스펙트럼 목록에서 정답 원소가 빨간 테두리로 강조됩니다.
• "정답 숨김" 버튼을 누르면 강조가 사라집니다.
💡 학습 팁
효과적인 학습 방법:
• 먼저 각 원소의 스펙트럼 패턴을 충분히 관찰하세요.
• 특징적인 선의 위치와 색깔을 기억하세요.
• 쉬운 문제(2개)부터 시작해서 점차 난이도를 높이세요.
• 틀린 문제는 "정답 표시"를 통해 복습하세요.
• 반복 연습을 통해 스펙트럼 패턴 인식 능력을 향상시키세요.

스펙트럼 특징 파악하기:
• 수소(H): 빨강, 청록, 남색, 보라색 4개의 선
• 헬륨(He): 노랑~주황 영역에 여러 선
• 나트륨(Na): 노란색 영역에 밝은 이중선이 특징
• 수은(Hg): 초록색 선이 매우 밝게 나타남
🔬 과학적 배경
원자가 에너지를 흡수하면 전자가 높은 에너지 준위로 이동하고, 다시 낮은 준위로 떨어질 때 특정 파장의 빛을 방출합니다. 각 원소는 고유한 에너지 준위 구조를 가지므로, 방출하는 빛의 파장도 원소마다 다릅니다. 이를 이용하면 멀리 있는 별의 구성 원소도 알아낼 수 있습니다!
